# Hospital-Analytics-System

## Project Overview

This is an end-to-end Hospital Analytics Project built using:

* Python
* Pandas
* NumPy
* PostgreSQL
* Power BI
* Excel Automation

The project analyzes healthcare data to generate:

* KPI reports
* business insights
* dashboards
* automated exports

---

## Project Features

### Data Cleaning

* Removed duplicates
* Cleaned text columns
* Converted dates
* Created Stay Duration KPI

### Python Analytics

* Revenue analysis
* Hospital performance
* Medical condition analysis
* Insurance analysis
* Trend analysis

### SQL Analytics

* KPI queries
* Window functions
* Ranking analysis
* Monthly trends

### Power BI Dashboard

* Executive dashboard
* Financial analysis
* Medical insights
* Operational analytics

### Automation

* Excel report exports
* Automated reporting
